# Odin Landing Page

A landing page built with **HTML** and **CSS** as part of **The Odin Project Foundations Course**.

## About

The goal of this project was to recreate a landing page design while practicing:

- HTML page structure
- CSS Flexbox
- Alignment and spacing
- Typography and colors
- Reusable CSS classes

## Technologies

- HTML5
- CSS3
- Flexbox
- Google Fonts

## What I Learned

During this project, I practiced:

- Breaking a page into multiple sections
- Using nested Flexbox containers
- Working with `justify-content`, `align-items` and `gap`
- Creating rows and columns
- Centering content
- Styling buttons, cards and navigation links

## Preview

The page contains:

- Navigation and hero section
- Information cards
- Testimonial section
- Call-to-action section
